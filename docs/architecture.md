# Architecture Overview

## Components:
1. **MCP Server** (main.py) - FastMCP server instance
2. **Auth Manager** (auth.py) - OAuth2 token handling
3. **Databricks Client** (client.py) - API wrapper
4. **Tools Module** (tools.py) - MCP tools definitions
5. **Resources Module** (resources.py) - MCP resources definitions
6. **Config Manager** (config.py) - Configuration handling
